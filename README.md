# 🍽️ Restaurant Menu & Order Management System (Jupyter Notebook)

## 📌 Overview

This project is created using Python in a Jupyter Notebook (`.ipynb`).
It simulates a basic restaurant system including menu handling, cart operations, inventory tracking, and sales analysis using core Python data structures.

## ✅ Task 1 — Explore the menu
